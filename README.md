# Transi Aracil — El Viaje hacia tu Origen

Landing page del proceso terapéutico individual de 10 sesiones de Transi Psicología Integral. Aquí está la descripción del producto, la metodología (recorrido cronológico por 10 etapas), el approach multi-herramienta, y la propuesta de entrevista de idoneidad gratuita.

## Stack

- HTML estático, CSS personalizado
- Sistema Liderar: paleta violeta `#7A3EFE` + dorado `#C6A87C` + cream `#FDFBF7`
- Tipografía: Playfair Display + Lato (Google Fonts)
- Iconos: Lucide (CDN)
- Sin build step

Subido a GitHub Pages. Ver `index.html` en la raíz.
